# Ejercicio: Agrupar Anagramas

## Descripción

Dado un arreglo de cadenas de texto, agrupa aquellas palabras que sean *anagramas* entre sí.

Un *anagrama* es una palabra formada al reorganizar las letras de otra, utilizando exactamente los mismos caracteres.

---
## Entrada

Un arreglo de strings:

```python
["eat", "tea", "tan", "ate", "nat", "bat"]
```
## Salida
```python
[
  ["eat", "tea", "ate"],
  ["tan", "nat"],
  ["bat"]
]
```
